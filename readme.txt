Name: Style Bandit
Version: 1.0
Authors: Mograph Mindset & Nate Lovell

https://aescripts.com/style-bandit/

//-------------------------------------------------------------------

This plugin extends functionality to Text Layers to let you quickly Copy and Paste text styles between layers.

To copy a text layer's properties, select a Text Layer and run Layer > Copy Text Style.

After the text style has been copied, it can be pasted to other layers with Layer > Paste Text Style.

Map to any Keyboard Shortcut!
ex: CTRL + ALT + C or ⌘ + ⌃ + C
