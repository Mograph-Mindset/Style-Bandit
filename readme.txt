This script extends functionality to Text Layers to quickly Copy and Paste text styles between layers.

The full plugin on aescripts wraps the script code in C++, and adds 2 new menu items to the After Effects Layer menu:
• Copy Text Style
• Paste Text Style

To copy a text layer's properties, select a Text Layer and run Layer > Copy Text Style.

After the text style has been copied, it can be pasted to other layers with Layer > Paste Text Style.

Map to any Keyboard Shortcut!
ex: CTRL + ALT + C or ⌘ + ⌃ + C
